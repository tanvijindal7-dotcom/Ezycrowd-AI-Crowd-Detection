# 🚦 Ezycrowd – AI Crowd Detection System

Ezycrowd is an AI-driven smart crowd detection and management system that leverages computer vision to monitor crowd density in real-time and generate alerts when overcrowding occurs. It aims to enhance safety and enable smarter surveillance in public spaces.

---

## 🚀 Features

* 🎯 Real-time crowd detection using computer vision
* 📊 Live dashboard for monitoring crowd density
* 🚨 Automated alerts when threshold is exceeded
* 📂 Upload images/videos for crowd analysis
* ⚡ Fast API-based backend for processing

---

## 🛠️ Tech Stack

* **Backend:** Python, FastAPI
* **Computer Vision:** OpenCV, YOLOv8
* **Frontend:** HTML, CSS, JavaScript
* **Tools:** Replit, GitHub

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Ezycrowd-AI-Crowd-Detection.git
cd Ezycrowd-AI-Crowd-Detection
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the backend

```bash
uvicorn main:app --reload
```

### 4. Open frontend

Open `index.html` in your browser
